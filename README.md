# 🥦 پلتفرم رژیمی هوشمند سیر (Seer Diet)

پلتفرم **سیر (Seer)** یک سیستم جامع و فول‌استک لوکس برای طراحی، طبخ و توزیع غذاهای رژیمی و ارگانیک به صورت کاملاً شخصی‌سازی شده برای ورزشکاران، بدنسازان و علاقه‌مندان به سبک زندگی سالم است. این سامانه با تلفیق علم تغذیه ورزشی و تکنولوژی‌های وب، تجربه‌ای بی‌نقص از کنترل کالری، مدیریت درشت‌مغذی‌ها (پروتئین، کربوهیدرات، چربی) و ارسال منظم لجستیک گرم را به ارمغان می‌آورد.

---

## 🚀 ویژگی‌های کلیدی سامانه (Core Features)

### ۱. طراح هوشمند رژیم (Smart Meal Planner)
* **محاسبه‌گر پیشرفته کالری (TDEE Calculator):** بر اساس اطلاعات فردی (قد، وزن، سن، جنسیت، ضریب فعالیت و هدف ورزشی).
* **دسته‌بندی اهداف بدنسازی:**
  * **کاهش وزن و چربی (چربی‌سوزی):** رژیم‌های کم‌کربوهیدرات با نسبت پروتئین بهینه.
  * **افزایش حجم و قدرت (عضله‌سازی):** پکیج‌های غنی از آمینواسیدها با کالری مازاد کنترل‌شده.
  * **تثبیت وزن و سلامت:** وعده‌های متوازن و ارگانیک غنی از آنتی‌اکسیدان‌ها.
* **اشتراک‌های انعطاف‌پذیر:** انتخاب ۳، ۵ یا ۷ روز در هفته به همراه وعده‌های صبحانه، ناهار، شام و میان‌وعده.

### ۲. پنل پیشرفته ورزشکاران (Athlete Interactive Dashboard)
* **ردیاب سلامت روزانه:** نمودارهای گرافیکی وضعیت کالری مصرفی، پروتئین، کربوهیدرات و چربی هفتگی.
* **مدیریت تحویل (Live Dispatch & Calendar):** نمایش وضعیت لجستیکی مرسوله‌ها (در حال آماده‌سازی، تحویل به پیک، تحویل داده شده).
* **انعطاف در تعویض وعده (Meal Swapping):** امکان جایگزینی غذای فردا با ده‌ها منوی جایگزین رژیمی تا ۲۴ ساعت قبل از تحویل.
* **کیف پول دیجیتال (Digital Wallet):** شارژ آسان حساب و کسر خودکار هزینه تمدید اشتراک.
* **سیستم ثبت تیکت پشتیبانی (CRM Tickets):** ارتباط مستقیم و ۲۴ ساعته با کارشناسان تغذیه و لجستیک.

### ۳. وبلاگ تغذیه ورزشی و علمی سیر سلامت (Seer Health Blog)
* شامل **۹ مقاله عمیق و تخصصی** در حوزه‌های رژیم‌های کتوژنیک، هیدراتاسیون، روزه‌داری متناوب، مزایای امگا ۳، سوپرفودهای دانه کامل و ریکاوری غدد عضلانی.
* نگارش شده توسط برترین پزشکان تغذیه ورزشی و سرآشپزهای رژیمی پلتفرم سیر.
* مجهز به سیستم فیلترینگ موضوعی پیشرفته و تخمین زمان مطالعه.

### ۴. پنل یکپارچه کادر سیر (Seer Staff Operations Command)
پنل مدیریت بر اساس نقش‌های سازمانی کاملاً تفکیک شده است:
* **مدیر کل (Super Admin):** نظارت بر درآمد کل، آمار توزیع پکیج‌ها، ثبت و ویرایش منوی غذاها و مدیریت دسترسی سایر همکاران.
* **سرآشپز ارشد (Chef/Cook):** مشاهده سفارشات فعال روز، دستور پخت‌های دقیق و تایید نهایی طبخ ارگانیک.
* **حسابدار ارشد (Accountant):** تحلیل تراکنش‌ها، نمودارهای مالی، درآمدهای دوره‌ای و تایید فاکتورها.
* **پشتیبانی مشترکین (CRM Agent):** پاسخ‌دهی به تیکت‌های ورزشکاران با استفاده از پاسخ‌های آماده پیشرفته یا پاسخ‌های سفارشی پزشک تغذیه.

## 🔒 امنیت اطلاعات و رمزنگاری داده‌ها (Database & Cryptographic Security)

جهت انطباق با حریم خصوصی ورزشکاران و جلوگیری از سرقت اطلاعات رژیمی، تماس و تراکنش‌های مالی، تمام داده‌های پایگاه داده محلی (پایداری روی مرورگر) با بهره‌گیری از یک **موتور رمزنگاری پیشرفته (Cryptographic Engine)** به آدرس `/src/lib/security.ts` محافظت می‌شوند:

* **مکانیزم رمزنگاری:** استفاده از الگوریتم ترکیبی **XOR Cipher با کلید نمک پویا (Dynamic Salting Key)** به همراه فشرده‌سازی و انکودینگ **Base64**.
* **محافظت خودکار (Secure Storage Wrapper):** توابع `secureStorage` به طور خودکار داده‌های حساس ورودی نظیر فاکتورها، آدرس‌ها، شماره تلفن‌ها، جزئیات پرداخت ورزشکاران و کلمات عبور کادر اداری را پیش از ذخیره در هارد دیسک/مرورگر رمزنگاری می‌کنند.
* **امنیت بصری:** در صورت باز کردن بخش Storage در مرورگر (DevTools)، هیچ‌گونه داده خوانا به صورت متن شفاف (Clear Text) وجود نداشته و اطلاعات به شکل کدهای رمزنگاری شده غیرقابل شکست نمایش داده می‌شوند.

## 🎨 معماری بصری و هویت طراحی (Design System)

دیزاین پلتفرم سیر از اصول **طراحی مینیمال و لوکس سوئیسی (Swiss-Modernist UI)** پیروی می‌کند:
* **تایپوگرافی معتبر:** استفاده از پکیج فونت‌های بدون نقص و با کنتراست بالا.
* **پالت رنگی ارگانیک:** پایه‌ای از رنگ‌های سنگی خنثی (`Stone 50/100`) همراه با رنگ نمادین **سبز زمردی ارگانیک (`Emerald 600/700`)** که نماد تازگی، سلامت و تمرینات پرانرژی است.
* **مجموعه آیکون‌های مدرن:** حذف کامل شکلک‌های سنتی (Emoji) و بازسازی همه‌جانبه بخش‌ها با استفاده از پکیج وکتور پیشرفته **Lucide-React** جهت جلوه بخشیدن به کلاسی حرفه‌ای و تجاری.
* **انیمیشن‌های هدفمند:** استقرار کتابخانه انیمیشن قدرتمند **Motion (Framer Motion)** جهت هدایت نگاه کاربر حین تغییر تب‌ها، باز شدن مودال‌ها و ثبت تراکنش‌ها با فیدبک‌های حرکتی روان ۶۰ فریم بر ثانیه.

## 🛠️ ساختار ماژولار پروژه (Directory & Architecture)

کد به گونه‌ای معماری شده تا خوانا، سبک و قابل نگهداری باشد و از متراکم‌سازی بی‌مورد پرهیز شده است:

```text
/
├── src/
│   ├── components/
│   │   ├── GarlicLogo.tsx        # هویت‌ساز و آیکون انحصاری لوگوی سیر به شکل سیر طلایی
│   │   ├── MealPlanner.tsx       # موتور هوشمند محاسبه‌گر کالری و پیکربندی اشتراک‌ها
│   │   ├── UserDashboard.tsx     # پنل شخصی ورزشکار (نمودارهای رژیمی، کیف پول، پیگیری لجستیک و تیکت‌ها)
│   │   ├── AdminPanel.tsx        # پنل مدیریت عملیات (سرآشپز، پشتیبانی، حسابدار و مدیر کل)
│   │   ├── BlogSection.tsx       # بخش بلاگ علمی و مدیریت مقالات پیشرفته
│   │   └── AuthModal.tsx         # سیستم احراز هویت و ورود ورزشکاران و همکاران
│   ├── lib/
│   │   └── security.ts           # ماژول رمزنگاری اطلاعات کاربران و رمزهای عبور
│   ├── assets/
│   │   └── images/               # گالری تصاویر لوکس عکاسی غذاهای رژیمی و تناسب اندام
│   ├── data/
│   │   ├── index.ts              # داده‌های ثابت منوی پایه و پلن‌های قیمت‌گذاری
│   │   └── blog.ts               # دیتابیس کامل ۹ مقاله تخصصی تغذیه ورزشی
│   ├── App.tsx                   # کامپوننت اصلی و هماهنگ‌کننده تعاملی سراسری پلتفرم
│   ├── types.ts                  # تایپ‌بندی و اینترفیس‌های تایپ‌اسکریپت به صورت کاملاً جامع
│   └── index.css                 # شیوه نامه‌های سراسری و ادغام فونت‌ها در Tailwind CSS
├── metadata.json                 # پیکربندی دسترسی‌ها و مجوزهای پلتفرم
├── package.json                  # مدیریت وابستگی‌ها و بسته‌های نرم‌افزاری
└── README.md                     # راهنمای جامع پلتفرم سیر رژیم
```

## 📦 نحوه راه‌اندازی و اجرای محلی (Installation & Usage)

برای راه‌اندازی پلتفرم در سیستم خود، مراحل زیر را دنبال کنید:

### ۱. نصب وابستگی‌ها
مطمئن شوید که Node.js بر روی سیستم شما نصب است. سپس در پوشه ریشه پروژه دستور زیر را وارد کنید:
```bash
npm install
```

### ۲. اجرای سرور توسعه دهنده (Dev Server)
برای اجرای نرم‌افزار در حالت توسعه محلی از دستور زیر استفاده کنید:
```bash
npm run dev
```
برنامه در آدرس `http://localhost:3000` آماده به کار خواهد بود.

### ۳. بررسی خطاهای کدنویسی (Linting)
جهت اطمینان از صحت استانداردهای تایپ‌اسکریپت و کدهای برنامه:
```bash
npm run lint
```

### ۴. ساخت نسخه نهایی برای انتشار (Production Build)
برای ایجاد کدهای فشرده و بهینه‌سازی شده نهایی جهت استقرار روی سرورهای ابری:
```bash
npm run build
```

## 💡 راهنمای کاربری برای تست سناریوها

1. **سناریوی ورزشکار:**
   * ابتدا با کلیک روی **ورود / ثبت‌نام** یک حساب کاربری ایجاد کنید.
   * به بخش **طراح هوشمند رژیم** رفته، قد، وزن و هدف بدنسازی خود را وارد کنید تا کالری و درشت‌مغذی‌های متناسب شما بلافاصله محاسبه شوند.
   * یکی از پکیج‌های اشتراکی را انتخاب کرده و زمان تحویل وعده‌ها را روی تقویم مشخص کنید.
   * با کلیک روی ثبت، اشتراک شما فعال شده و سفارشات زمان‌بندی شده در **داشبورد من** ظاهر می‌شوند.
   * اکنون می‌توانید کیف پول خود را شارژ کنید، منوی فردای خود را با غذای رژیمی دیگری تعویض نمایید یا تیکت پشتیبانی ارسال کنید.

2. **سناریوی همکاران سیر (Admin Operational Flow):**
   * در منوی پایین سایت (Footer) یا هدر، روی دکمه **ورود همکاران** کلیک کنید.
   * یکی از پرسنل پیش‌فرض را انتخاب کنید (مثلاً آقای عباسی به عنوان سرآشپز یا خانم سلیمانی به عنوان مدیر کل).
   * پس از ورود، بلافاصله منوی **پنل کادر سیر** در هدر اصلی سایت ظاهر می‌شود.
   * با کلیک بر روی آن وارد کنترل پنل فوق‌پیشرفته و هوشمند سیر شوید و سفارشات، دستور پخت، نمودارهای مالی و سیستم CRM پشتیبانی را بررسی و کنترل کنید.
